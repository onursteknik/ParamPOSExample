Param POS Entegrasyon Örnek Proje

```
* Bu projede ParamPOS(eski adıyla TurkPOS) sanal pos entegrasyonu örneği gösterilmiştir.

* ParamPOS sanal pos entegrasyonu yapmak zorunda kaldığım bir projede
ParamPOS'un -http://dev.param.com.tr/tr/api/odeme- API dokümantasyonunu
ve verdiği .NET proje örneğini beğenmediğim,tam anlamıyla açıklayıcı
olmadığını düşündüğüm için, bu projeyi oluşturmaya karar verdim.

* Projenizde kullanmak için bu örnek projeden işinize yarayacak kısımları
dilediğinizce alabilirsiniz.

* Elimden geldiğinde satır aralarında açıklama yapmaya çalıştım.
Yanlış yazdığım, yanlış bildiğim yerler varsa ise şimdiden özür diler,yanlış olan yerleri
düzeltmenizi rica ederim.

```
